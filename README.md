# SamvaadSetu

### Install the required packages for virtual environment creation

```python
pip install venv
```

### Create a Python Environment

```python
python -m venv env
```

### Activate the Environment

- Windows

```cmd
env\Scripts\activate
```

- Linux

```bash
source env/bin/activate
```

### Inside the Environment install the required python packages

```python
pip install -r requirements.txt
```
